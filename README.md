## About

This repo contains a Nix Flake for running the Project Horus Telemetry Decoder "horus-gui": [https://github.com/projecthorus/horus-gui](https://github.com/projecthorus/horus-gui)

## Current Version

horus-gui v0.6.1

## Usage

```bash
nix run git+https://codeberg.org/robinfinkelmann/horus-gui-nix
```
